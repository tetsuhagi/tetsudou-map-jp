本サイトについて

最終更新: 2026年5月

「鉄道マップ全国版」は、日本国内の主要な特急・新幹線・寝台特急の運行スケジュールを参考に、 現在時刻における列車のおおよその走行位置を地図上にシミュレーション表示する Web サイトです。 完全静的なフロントエンドのみで動作し、サーバー側で運行情報を取得しているわけではありません。

何ができるサイトか

  • 全国の主要な特急・新幹線・寝台特急（合計47路線）の走行イメージを地図上で確認できます
  • 平日／土日祝のダイヤを曜日に応じて自動切替します
  • 「ナイトモード」を有効化すると、時間帯（朝・昼・夕・夜）に応じて地図の色味が変化します
  • スマートフォン・タブレット・PC のいずれにも対応しています

仕組み

各列車について、おおよその発着時刻と停車駅をまとめた独自の概略データを用意しています。 ブラウザ上のJavaScriptが現在時刻を参照し、その時点で列車がどの区間を走っているかを線形補間で算出して、 地図上のアイコンとして表示する仕組みです。

地図は OpenStreetMap のタイルを、 路線形状は国土交通省「国土数値情報（鉄道データ）」を加工して使用しています。 リアルタイム運行情報のAPI接続は一切行っていないため、 遅延・運休・臨時列車・運転変更などは反映されません。

想定する楽しみ方

  • 寝台特急サンライズ瀬戸／出雲が深夜に瀬戸内・山陰を渡っていく様子を眺める
  • 旅行前に「新幹線でどの時間にどこまで行けそうか」を視覚的にイメージする
  • 鉄道好きの方の地図上シミュレーションとして気軽に楽しむ

運営について

本サイトは個人が趣味として運営する非公式サイトです。 いかなる鉄道事業者・公的機関とも関係ありません。 運営者は鉄道会社の関係者ではなく、表示している情報も鉄道会社から提供されたものではありません。

ご注意: 本サイトの表示は実際の運行情報ではなく、概略データに基づくシミュレーションです。 実際の運行情報・最新の時刻表は、各鉄道事業者の公式サイトをご確認ください。

使用技術

  • HTML / CSS / JavaScript（ES Modules）の完全静的サイト
  • 地図ライブラリ: Leaflet
  • 地図タイル: OpenStreetMap contributors
  • 路線形状データ: 国土交通省「国土数値情報（鉄道データ）」を加工して作成
  • ホスティング: Cloudflare Pages
