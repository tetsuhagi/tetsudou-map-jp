東海道・山陽新幹線に最上級「Supreme Class」導入｜2026年10月に23年ぶり個室復活・料金と発売日を整理

公開: 2026年6月19日 / カテゴリ: 鉄道ニュース

💡 まずはここだけ！3つの要点

  • JR東海・JR西日本が東海道・山陽新幹線（N700S・16両編成）に最上級クラス「Supreme Class」を導入。2026年10月1日にまず個室タイプ「Cabin」から
  • 7号車（2名まで利用可）・10号車（1名）に各1室の電子錠付き個室。東海道新幹線では2003年以来、23年ぶりの個室復活
  • 半個室タイプ「Seat」は2027年度中に開始予定。Cabinの発売は2026年9月15日（エクスプレス予約・スマートEX限定）
🙋‍♂️
新幹線にガチの個室て、ロマンの塊やん。一回乗ってみたいぞ

JR東海とJR西日本は、東海道・山陽新幹線に新たな最上級座席クラス 「Supreme Class（スプリームクラス）」 を導入すると発表しました。 まず2026年10月1日に個室タイプ「Cabin」が登場し、東海道新幹線では23年ぶりに個室が復活します。 この記事では、発表で明らかになった設備・料金例・発売スケジュールのポイントを整理します。

ご注意: 本記事は2026年6月19日時点で公開されている情報をもとに作成しています。 設備・料金・発売スケジュール・対象列車等は今後変更される可能性があります。 最新・正確な情報は JR東海「Supreme Class」公式ページ などの一次情報でご確認ください。

「Supreme Class（スプリームクラス）」とは

Supreme Class は、東海道・山陽新幹線にこれまでなかった グリーン車を上回る最上級クラス として新設されるサービスです。 コンセプトは「プライベートな空間で、あなただけの上質な移動時間を」。 座席という枠を超えて、移動そのものを特別な体験にすることを狙ったクラスと位置づけられています。

Supreme Class には2つのタイプが用意されます。完全な個室の 「Cabin（キャビン）」 と、 半個室の 「Seat（シート）」 です。 このうち先行して登場するのが Cabin で、2026年10月1日からサービスを開始します。

導入の概要

  • 運行会社: JR東海・JR西日本
  • 対象路線: 東海道・山陽新幹線
  • 対象車両: N700S（16両編成）
  • 設定号車: 7号車・10号車に各1室
  • サービス開始: 2026年10月1日（個室タイプ「Cabin」）／半個室「Seat」は2027年度中の予定

対象は16両編成のN700Sで運行する列車です。どの列車に設定されるかなど詳細は、公式の案内で順次示される見込みです。

東海道新幹線の個室、2003年以来って…復活めっちゃ久しぶりなんやな

個室「Cabin」の設備

Cabin は、N700Sの 7号車に1室・10号車に1室 が設けられます。 号車によって定員と設備が異なるのが特徴です。

  • 7号車: 2名まで利用できる個室。ソファ やスーツケースを置ける荷物スペースも備える、ゆとりのあるタイプ
  • 10号車: 1名利用の個室。ひとりの移動時間を上質に過ごすためのタイプ

共通する主な設備は次のとおりです。

  • 電子錠付きの扉 による、プライベート感とセキュリティの確保
  • レッグレスト付きのリクライニングシート
  • 専用 Wi-Fi・大形テーブル
  • 専用タブレットで 照明・空調・放送を自分好みに調整 可能
  • 世界初という「シートスピーカー」 を搭載

照明や空調まで手元のタブレットでコントロールできる点は、従来の新幹線の座席にはなかった発想です。 「個室＋自分仕様にできる空間」という方向に振り切ったクラスといえます。

公式が示した料金の例は次のとおりです（おとな1名・片道・通常期・エクスプレス予約の場合の目安）。

東京〜新大阪 60,500円 42,100円
東京〜博多 90,220円 63,620円

料金は時期（閑散期・繁忙期・最繁忙期）によって変動します。 7号車は2名まで利用できる個室のため、利用人数や予約方法による料金の適用条件は、購入時に公式の案内でご確認ください。 グリーン車をさらに上回る価格帯で、「ここぞという旅・出張」に向けた特別なクラスという位置づけがうかがえます。

半個室「Seat」は2027年度中に

Supreme Class のもう一方のタイプ、半個室の「Seat」は、2027年度中のサービス開始を予定しています。 Cabin と同程度のサービスを提供するとされており、完全個室よりは手が届きやすい選択肢として期待されます。 まずは Cabin が先行し、ラインアップが順に広がっていく形です。

発売・体験乗車

  • Cabin の発売: 2026年9月15日から（エクスプレス予約・スマートEX限定）。発売時刻など詳細は公式の案内を確認
  • 体験乗車: 2026年7月25日・26日に実施予定。応募の締切は6月30日とされています（募集状況は公式で確認を）

予約はエクスプレス予約・スマートEXに限定される点に注意が必要です。 各号車に1室しかない希少な席のため、発売直後は相当な競争になることが予想されます。 狙っている人は、予約サービスの登録など事前準備を済ませておくと安心でしょう。

23年ぶりの個室復活という意味

東海道新幹線では、かつて100系などに個室が設けられていましたが、2003年を最後に姿を消していました。 Supreme Class の Cabin は、東海道新幹線として2003年以来、23年ぶりの個室復活にあたります。

近年は、グリーン車のさらに上を行く上級クラスへの需要が各社で意識されています。 寝台特急 サンライズ出雲・サンライズ瀬戸 の個室がいつも人気で、 新幹線でも「ただ速く移動する」だけでなく「移動時間そのものを上質に過ごしたい」というニーズが確実にあります。 Supreme Class は、その需要に新幹線の最上位クラスとして応える試みといえます。

7号車は2人用でソファ付き、10号車は1人用かぁ。おひとりさま出張で10号車、地味に贅沢やな
